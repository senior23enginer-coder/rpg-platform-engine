======================================================================
CAPITULO 1 — CONCEPTO DEL SISTEMA
======================================================================

Este juego adapta la experiencia de Fallout 4 a mesa como una combinación de:

| Capa | Qué aporta |
|---|---|
| Wargame táctico | Posición, nodos, cobertura, línea de visión, alcance, control de zona y daño por partes. |
| TTRPG | Diálogo, narración, decisiones, facciones, consecuencias, libertad y exploración narrativa. |
| RPG por turnos | AP, acciones, perks, experiencia, niveles, inventario, equipo y progresión. |
| 2d20 | Tiradas SPECIAL + habilidad, dificultad, éxitos, Momentum y riesgo. |

Regla madre:
Todo personaje, NPC, criatura, robot o enemigo ocupa un nodo cuando la escena requiere precisión táctica.
Moverse, atacar, explorar, interactuar, cubrirse o usar equipo consume AP cuando existe presión, peligro,
combate, sigilo, exploración activa o turno estructurado.

Cuando no existe peligro, presión o consecuencia importante, el Director o el modo sin GM puede resolver
la escena de forma narrativa sin exigir AP por cada paso.

CAPITULO 2 — MODOS DE JUEGO
======================================================================

| Modo | Qué representa | Uso de AP |
|---|---|---|
| Campaña | Historia principal adaptada, protagonistas de Vault 111 y facciones base. | AP activo en combate, exploración, persecución, sigilo y escenas con presión. |
| Modo Libre | Supervivientes propios, campaña emergente, contratos, rutas y asentamientos. | AP activo cuando el mundo se juega por nodos o hay riesgo. |

El Tomo 01 permite jugar cualquiera de los modos de forma básica. Si la mesa quiere detalle completo de
supervivencia, asentamientos, armas, armaduras, enemigos o ubicaciones, se consulta el tomo especializado.

CAPITULO 4 — PERKS, HABILIDADES, EXPERIENCIA Y NIVELES
======================================================================

4.1 Tipos de perks

| Tipo de perk | Cómo funciona | Consume AP |
|---|---|---|
| Pasivo | Siempre está activo si cumple requisito. | No. |
| Activo | El jugador decide usarlo durante una acción. | Sí, si modifica una acción o crea una acción especial. |
| Reacción | Se activa ante un disparador: recibir daño, esquivar, oportunidad, detectar, resistir. | Puede consumir AP, reacción o uso limitado. |
| Mejora de acción | Reduce coste, baja dificultad, suma daño, mejora alcance o agrega efecto. | Consume el AP de la acción modificada. |
| Perk de escena | Afecta viaje, comercio, diálogo, asentamiento, crafteo o exploración larga. | Normalmente no usa AP de combate; usa escena/tiempo. |

4.2 Activación de perks

| Caso | Regla |
|---|---|
| Perk pasivo | Aplicar automáticamente si el personaje cumple requisito. |
| Perk activo en combate | Declarar antes de tirar, pagar AP si aplica y resolver efecto. |
| Perk de exploración | Declarar durante revisión, búsqueda, hackeo, ganzúa, rastreo o interacción. |
| Perk de reacción | Activar cuando ocurre el disparador. Si hay duda, el Director decide antes de tirar daño. |
| Perk con usos limitados | Marcar uso por combate, escena, día o descanso según ficha. |

4.3 Experiencia

La experiencia se gana por objetivos, exploración, combate, negociación, descubrimientos y consecuencias.
No se recomienda dar XP solo por matar; el sistema premia resolver escenas.

| Fuente de XP | XP sugerida |
|---|---:|
| Descubrimiento menor, pista, ruta segura | 10-25 XP |
| Superar obstáculo, hackeo, cerradura, trampa o peligro | 25-50 XP |
| Combate menor o amenaza baja | 25-75 XP |
| Combate medio o escena peligrosa | 75-150 XP |
| Objetivo de misión completado | 100-300 XP |
| Misión completa | 300-800 XP |
| Decisión narrativa importante | 100-500 XP |
| Resolver escena sin violencia cuando era posible | 50-250 XP |
| Defender asentamiento o ciudad | 150-600 XP |

4.4 XP necesaria para subir de nivel

Regla base propuesta para I6.1:

| Nivel nuevo | XP total necesaria |
|---:|---:|
| 2 | 200 |
| 3 | 500 |
| 4 | 900 |
| 5 | 1400 |
| 6 | 2000 |
| 7 | 2700 |
| 8 | 3500 |
| 9 | 4400 |
| 10 | 5400 |

Formula opcional para niveles superiores:
XP total para siguiente nivel = XP actual de nivel anterior + (Nivel nuevo x 100).

4.5 Punto de nivel

Al subir de nivel, el personaje gana 1 punto de nivel.

| Uso del punto de nivel | Regla |
|---|---|
| Subir SPECIAL | Aumenta 1 atributo en +1, sin pasar el máximo permitido por etapa de campaña. |
| Comprar perk nuevo | Debe cumplir requisito SPECIAL/nivel si existe. |
| Subir rango de perk | Debe tener rango anterior y cumplir requisito. |
| Guardar punto | Permitido si la mesa quiere comprar perk más adelante. |

Regla central:
Un punto de nivel se gasta en SPECIAL o perk/rango de perk, no en ambos.

CAPITULO 5 — TURNOS, AP Y CATEGORIAS DE ACCIONES
======================================================================

5.1 Categorías oficiales

Desde I6.1 solo existen tres categorías maestras de acción:

| Categoría | Qué cubre | Ejemplos |
|---|---|---|
| Movimiento | Desplazarse o cambiar posición en el entorno. | Moverse a nodo, correr, saltar, subir, bajar, trepar, sigilo, huir. |
| Combate | Atacar, defender, usar armas, cubrirse, granadas y habilidades ofensivas. | Disparar, golpear, atacar sin armas, cobertura, granada, apuntar, perk de combate. |
| Exploración | Descubrir, registrar, usar herramientas, interactuar, curar, reparar, recargar o manipular el mundo. | Revisar entorno, loot, cerradura, hackeo, leer nota, recargar, Stimpak, reparar robot. |

Las acciones antes llamadas tareas quedan absorbidas por Exploración/Interacción.

5.2 Tabla maestra de AP

| Tipo de acción | Coste AP recomendado | Ejemplo |
|---|---:|---|
| Movimiento mínimo | 1 AP | Cambiar posición dentro del nodo. |
| Movimiento normal | 1 AP | Moverse a 1 nodo conectado normal. |
| Movimiento difícil | 2 AP | Escombros, puerta pesada, subir/bajar escalera. |
| Movimiento rápido | 3 AP | Moverse a 2 nodos conectados de forma controlada. |
| Carrera | 4 AP | Correr hasta 2 nodos, genera ruido/exposición. |
| Exploración simple | 1-2 AP | Mirar, escuchar, leer nota, interactuar simple. |
| Exploración media | 2-3 AP | Buscar loot, revisar entorno, usar objeto, recargar. |
| Exploración compleja | 4-5 AP | Hackear, reparar, abrir cerradura difícil, desactivar trampa. |
| Ataque ligero | 3 AP | Pistola, arma ligera, melee básico. |
| Ataque medio | 4 AP | Rifle, granada, melee pesado. |
| Ataque pesado | 5 AP | Arma pesada, acción peligrosa o ataque fuerte. |
| Acción prolongada | Escena | Reparación grande, construcción, cirugía, viaje seguro. |

6.2 Regla de ubicación explorable

Cada ubicación explorable tiene un submapa con nodos.
Una ubicación controlada, ciudad segura o asentamiento gestionado no exige AP para caminar normalmente.
Vuelve a consumir AP si ocurre combate, ataque, sigilo, infiltración, persecución, exploración activa o evento.

| Estado de ubicación | AP para recorrer | Ejemplo |
|---|---|---|
| Explorable | Sí | Concord hostil, Vault desconocido, ruina nueva. |
| Controlada | No en recorrido normal | Sanctuary limpiada. |
| Asentamiento | No en gestión normal | Construir, comerciar, producir recursos. |
| Asentamiento atacado | Sí | Defensa por nodos. |
| Ciudad segura | No en movimiento social | Diamond City en calma. |
| Ciudad en conflicto | Sí | Ataque, infiltración, persecución. |
| Ruta segura | No o escena | Camino despejado y conocido. |
| Ruta peligrosa | Sí o evento | Emboscada, clima, patrulla, criatura. |

CAPITULO 7 — COMBATE TACTICO POR NODOS
======================================================================

7.1 Secuencia de ataque

| Paso | Acción |
|---:|---|
| 1 | Declarar atacante, arma, objetivo y nodo objetivo. |
| 2 | Contar distancia en nodos. |
| 3 | Confirmar línea de visión. |
| 4 | Confirmar alcance del arma. |
| 5 | Determinar si el defensor tiene cobertura válida. |
| 6 | Tirar SPECIAL + habilidad contra dificultad. |
| 7 | Si impacta, determinar parte del cuerpo afectada. |
| 8 | Calcular daño bruto. |
| 9 | Restar cobertura aplicable, defensa de equipo y resistencias. |
| 10 | Aplicar daño final a PV y efecto local de parte del cuerpo. |
| 11 | Aplicar ruido, estados, sangrado, derribo o reacción enemiga. |

7.2 Cobertura, defensa y distancia

La cobertura no es armadura. Protege porque bloquea, reduce o dificulta la línea de tiro.
La elevación no es cobertura; es condición táctica de ángulo, visión y alcance.

| Tipo de cobertura | Ejemplos | DEF a 1 nodo | DEF a 2 nodos | DEF a 3 nodos | DEF a 4+ nodos | Notas |
|---|---|---:|---:|---:|---:|---|
| Sin cobertura | Calle abierta, plaza, pasillo limpio | +0 | +0 | +0 | +0 | No protege. |
| Ligera | Árbol delgado, puerta rota, caja pequeña, mueble débil, cerca baja | +1 | +1 | +2 | +2 | Débil de cerca; útil a distancia. |
| Media | Auto abandonado, muro bajo, mostrador, pila de cajas, chatarra compacta | +1 | +2 | +3 | +3 | Defensa estándar. |
| Pesada | Contenedor, muro grueso, barricada sólida, esquina, columna grande | +2 | +3 | +4 | +4 | Muy fuerte contra disparo frontal. |
| Total | Pared completa, interior cerrado, esquina sin línea directa, puerta sólida | Bloquea | Bloquea | Bloquea | Bloquea | No hay ataque directo salvo flanqueo, destrucción, apertura o explosivo. |
| Destructible | Puerta, madera, caja, barricada débil, auto dañado | Según material | Según material | Según material | Según material | Usa ligera/media/pesada pero puede romperse. |
| Peligrosa | Auto con combustible, barril, generador, reactor, tanque | +1 a +3 | +1 a +3 | +1 a +3 | +1 a +3 | Protege, pero puede explotar o causar efecto. |

7.3 Flanqueo y elevación

| Situación | Efecto |
|---|---|
| Ataque frontal | Aplica cobertura completa. |
| Ataque lateral parcial | Reduce cobertura en 1 DEF. |
| Retaguardia o flanqueo claro | Niega cobertura ligera/media. |
| Cobertura total flanqueada | Deja de bloquear si aparece línea clara. |
| Ataque desde elevación con buen ángulo | Reduce cobertura en 1 DEF si la ficción lo permite. |

7.4 Cobertura contra tipos de ataque

| Ataque | Cómo interactúa con cobertura |
|---|---|
| Balístico directo | Aplica cobertura si está entre atacante y defensor. |
| Energía directa | Aplica cobertura; algunos efectos pueden dañar cobertura si el arma lo indica. |
| Cuerpo a cuerpo | La cobertura solo importa si impide entrar al mismo nodo o bloquea paso. |
| Escopeta a 1 nodo | Cobertura ligera no basta si hay línea clara; cobertura media/pesada sí aplica. |
| Explosivo | Apunta a nodo; cobertura frontal puede no proteger si el explosivo cae dentro o detrás. |
| Molotov/fuego | Cobertura no inflamable protege mejor; cobertura inflamable puede prenderse. |
| Gas/radiación | La cobertura física no protege salvo sellado, máscara, servoarmadura o refugio. |

Regla:
Una cobertura destructible tiene DEF mientras existe. Si recibe daño acumulado igual o superior a sus PV de cobertura, baja un grado o se destruye.

| Cobertura destructible | Cobertura equivalente | PV cobertura | Al destruirse | Notas |
|---|---|---:|---|---|
| Caja pequeña / mueble débil | Ligera | 6 PV | Desaparece | Rompible con disparos, melee o explosivo menor. |
| Puerta de madera / cerca simple | Ligera | 8 PV | Se abre/rompe | Puede seguir como obstáculo narrativo. |
| Barricada improvisada | Media | 14 PV | Baja a ligera o se rompe | Hecha con tablas, basura o muebles. |
| Pila de cajas / chatarra suelta | Media | 16 PV | Baja a ligera | Puede dejar loot o terreno difícil. |
| Auto abandonado muy dañado | Media | 22 PV | Baja a peligrosa o arde | Si tiene combustible puede explotar. |
| Auto abandonado completo | Media/Pesada | 30 PV | Baja a media o peligrosa | Buen nodo de cobertura urbana. |
| Contenedor metálico delgado | Pesada | 36 PV | Baja a media | Armas pesadas/explosivos lo dañan mejor. |
| Muro de ladrillo parcial | Pesada | 42 PV | Abre línea parcial | Puede crear escombros. |
| Barricada sólida / saco de arena | Pesada | 45 PV | Baja a media | Difícil de destruir con armas ligeras. |
| Generador / barril / tanque peligroso | Peligrosa | 12-30 PV | Explota o libera efecto | Usar tabla de peligro de escena. |

Reglas de daño a cobertura:

| Ataque | Efecto contra cobertura |
|---|---|
| Arma ligera | Daño normal solo si la ficción permite dañar el material. |
| Arma media/rifle | Daño normal contra madera, muebles, puertas y chatarra ligera. |
| Arma pesada | Daño completo contra casi toda cobertura destructible. |
| Explosivo | Daño completo al nodo de cobertura; puede afectar nodos adyacentes. |
| Fuego/molotov | Daña materiales inflamables; poco efectivo contra metal o piedra. |
| Melee | Solo contra cobertura cercana y material razonable. |

7.6 Cálculo de daño

Formula core:
Daño final = Daño bruto del arma + bonos - DEF de cobertura aplicable - DEF/Resistencia de pieza equipada - reducción por perk/estado.

Reglas:
- El daño final nunca baja de 0.
- Si impacta pero el daño final es 0, puede causar ruido, presión, pérdida de cobertura, retroceso o daño superficial si la escena lo justifica.
- Primero se aplica cobertura contra el impacto; si el ataque supera la ficción de la cobertura, se aplica defensa de equipo.

7.7 Tipos de defensa

| Defensa | Protege contra |
|---|---|
| DEF física / DR | Balas, golpes, mordidas, fragmentos, melee. |
| DEF energética / ER | Láser, plasma, electricidad, energía. |
| DEF radiación / RR | Radiación, armas rad, ambiente radiactivo. |
| DEF explosiva | Explosivos, granadas, minas, misiles. |
| Cobertura | Ataques a distancia bloqueados por objeto/posición. |

7.8 Partes del cuerpo humanoide

Todo humanoide estándar tiene seis partes de cuerpo. Estas partes se usan para saber qué pieza de armadura protege, qué efecto local puede aparecer y qué acciones quedan afectadas.

| 1d6 | Parte | Protección asociada | Acciones afectadas si queda dañada |
|---:|---|---|---|
| 1 | Cabeza | Casco, máscara, capucha reforzada, pieza de cabeza. | Percepción, puntería, habla, visión, concentración. |
| 2 | Torso | Pechera, ropa reforzada, pieza de torso. | PV principal, respiración, resistencia, sangrado, estabilidad. |
| 3 | Brazo izquierdo | Brazo izquierdo de armadura, guante, manga reforzada. | Armas a dos manos, bloqueo, herramientas, recarga, fuerza aplicada. |
| 4 | Brazo derecho | Brazo derecho de armadura, guante, manga reforzada. | Ataque, puntería, recarga, hackeo manual, melee, uso de herramientas. |
| 5 | Pierna izquierda | Pierna izquierda de armadura, bota, rodillera. | Movimiento, correr, saltar, sigilo, equilibrio, trepar. |
| 6 | Pierna derecha | Pierna derecha de armadura, bota, rodillera. | Movimiento, correr, saltar, sigilo, equilibrio, trepar. |

7.9 Determinación de parte impactada

La parte impactada se define después de confirmar que el ataque impacta. La tirada 2d20 no determina la parte del cuerpo; el 2d20 determina si el ataque acierta y con cuántos éxitos. La parte se determina con posición, visibilidad, cobertura, tipo de ataque o una tirada 1d6.

| Situación | Cómo se determina la parte impactada |
|---|---|
| Ataque normal sin apuntar | Tirar 1d6 en la tabla de partes del cuerpo. |
| Ataque apuntado | El atacante declara la parte antes de tirar. Aumenta la dificultad en +1 o exige una condición favorable de línea de visión. |
| Ataque con Momentum | Si el ataque impacta y genera Momentum suficiente, el atacante puede elegir una parte visible en lugar de tirar 1d6. |
| Objetivo en cobertura parcial | Solo pueden impactarse partes visibles. Si el 1d6 marca una parte cubierta, se repite la tirada entre partes visibles o el ataque golpea la cobertura. |
| Objetivo en cobertura total | No hay parte impactada porque no existe línea de ataque directa. Se debe flanquear, destruir la cobertura, abrir una ruta o usar área/explosivo. |
| Ataque desde elevación | Puede permitir elegir entre partes superiores visibles: cabeza, torso o brazos, si la línea de visión lo permite. |
| Ataque cuerpo a cuerpo | Normalmente impacta torso o extremidad. El atacante puede elegir parte visible con +1 dificultad. |
| Ataque por la espalda | Puede elegir torso o una extremidad visible si el objetivo no puede defenderse o no detectó el ataque. |
| Explosivo o área | Impacta el nodo. El objetivo recibe daño en torso y una parte aleatoria adicional si el daño final es relevante. |
| Fuego, gas o radiación | No siempre usa parte concreta; puede afectar torso/sistema general salvo que una pieza o máscara proteja. |
| Robot, criatura o enemigo no humanoide | Usa la tabla equivalente del Tomo 03. Si no existe, usar: cabeza/sensor, núcleo/torso, apéndices/armas, patas/movilidad. |

7.10 Tabla de partes visibles según cobertura

| Cobertura del objetivo | Partes normalmente visibles |
|---|---|
| Sin cobertura | Todas las partes pueden ser impactadas. |
| Cobertura ligera | Cabeza, torso parcial, brazos o piernas según objeto. |
| Cobertura media | Cabeza, brazos y parte superior del torso suelen ser visibles. Piernas pueden quedar cubiertas. |
| Cobertura pesada | Normalmente solo cabeza, brazo de disparo o una parte expuesta. |
| Cobertura total | Ninguna parte visible para ataque directo. |
| Cuerpo a tierra | Cabeza, brazos y torso parcial; piernas menos expuestas si hay obstáculo. |
| Elevación del atacante | Puede revelar cabeza, torso y brazos detrás de coberturas bajas. |
| Flanqueo claro | La cobertura puede dejar de proteger la parte que antes estaba cubierta. |

7.11 Estados por daño local

| Daño final en una parte | Estado local | Efecto de juego |
|---:|---|---|
| 1-2 | Rasguño / golpe menor | Sin penalización mecánica estable; puede dejar marca narrativa. |
| 3-5 | Herida | +1 dificultad a acciones que usen esa parte hasta recibir curación básica o descanso seguro. |
| 6-8 | Parte dañada | +1 AP para acciones que dependan de esa parte o +1 dificultad, según la acción. |
| 9+ | Parte crítica temporal | La parte pierde función durante la escena o hasta curación avanzada; puede causar caída, desarme, sangrado o incapacidad temporal. |

7.12 Daño con equipo

| Caso | Cómo se calcula |
|---|---|
| Sin equipo en la parte | Solo resta cobertura, perks o resistencia general si aplica; la parte recibe el daño restante. |
| Con ropa ligera | Resta la DEF de la prenda si cubre esa parte. |
| Con armadura por pieza | Resta la DEF de la pieza correspondiente: cabeza, torso, brazo izquierdo, brazo derecho, pierna izquierda o pierna derecha. |
| Con servoarmadura | Resta la DEF de la pieza de servoarmadura; la pieza puede recibir daño antes que el piloto. |
| Cobertura + armadura | Primero se aplica cobertura contra el ataque; si el ataque pasa, se aplica la defensa de la pieza impactada. |
| Ataque apuntado a parte sin protección | Ignora piezas que no cubren esa parte. |

Ejemplo:
Un raider dispara desde 2 nodos con rifle de tubo. Daño bruto 7. El jugador está en cobertura media a 2 nodos (+2 DEF) y lleva pechera con DEF física 2. La parte afectada es torso.
Daño final = 7 - 2 cobertura - 2 pechera = 3 PV. El torso recibe una herida.

7.13 Alcance de armas por nodos

|---|---:|---|---|
| Sin armas | 0 | Puño, patada, garra, mordida | Requiere mismo nodo. |
| Cuerpo a cuerpo corto | 0 | Cuchillo, machete, porra | Mismo nodo. |
| Cuerpo a cuerpo largo | 0-1 | Lanza, garra enorme, arma improvisada larga | Solo si arma/criatura lo permite. |
| Escopeta corta | 1 | Recortada, doble cañón cercana | Fuerte cerca; pierde eficacia lejos. |
| Pistola | 1-2 | 10mm, pistola de tubo | Interiores y calles cortas. |
| Subfusil/automática corta | 1-2 | Subfusil, pistola automática | Puede usar supresión. |
| Rifle corto/láser común | 2-3 | Rifle de tubo, rifle láser | Rango medio estándar. |
| Rifle largo | 3 | Rifle de combate, rifle de caza | Bueno en calles abiertas. |
| Francotirador | 4+ | Rifle francotirador | Requiere línea clara. |

8.1 Inventario

El inventario se divide en:

| Zona de inventario | Qué contiene |
|---|---|
| Equipo equipado | Armas listas, armadura puesta, ropa, casco, herramientas a mano. |
| Mochila / carga | Objetos transportados pero no listos. |

| Chatarra / materiales | Recursos para reparar, construir o comerciar. |
| Munición | Balas, células, cartuchos, misiles, núcleos. |
| Chapas | Moneda; no pesan salvo regla opcional. |

8.2 Peso y carga

Regla base:
La carga máxima depende de STR y equipo. Si la mesa no quiere cálculo duro, usar tres estados.

| Estado de carga | Condición | Efecto |
|---|---|---|
| Ligera | Carga cómoda | Sin penalización. |
| Pesada | Mucho equipo, armas o chatarra | +1 dificultad a correr, sigilo, trepar o nadar; puede reducir AP en escenas largas. |
| Sobrecargada | Excede capacidad razonable | No puede correr; movimiento difícil cuesta +1 AP; sigilo casi imposible. |

8.3 Munición

| Regla | Aplicación |
|---|---|
| Disparo simple | Consume 1 munición del tipo correcto. |
| Ráfaga corta | Consume 3 municiones o unidad equivalente. Puede mejorar daño/supresión. |
| Ráfaga larga / automática | Consume 5+ municiones; fuerte ruido y posible desperdicio. |
| Recarga sencilla | 2 AP. |
| Recarga pesada/compleja | 3 AP. |
| Sin munición | No puede disparar salvo arma alternativa, melee o acción de buscar/cambiar equipo. |
| Munición rara | Debe controlarse por tipo: energía, misil, mini nuke, núcleos, etc. |

8.4 Chapas

Las chapas se ganan por:

| Fuente | Chapas sugeridas |
|---|---:|
| Loot menor | 1-10 |
| Contenedor común | 5-25 |
| Saqueador/enemigo humano | 5-40 |
| Recompensa pequeña | 25-100 |
| Trabajo local | 50-200 |
| Misión completa | 100-500+ |
| Venta de armas/equipo | Según valor y tienda. |
| Comercio de chatarra | Bajo, salvo materiales raros. |

Regla:
Las chapas son recompensa económica, no sustituyen XP. Una misión puede dar XP, chapas, reputación, acceso a tienda, aliado o asentamiento.

Unidad usada:
- UP = unidad de peso del sistema, equivalente a la lectura de peso usada por armas, armaduras, munición y objetos.
- Capacidad máxima core = 200 + STR x 10 UP.

Penalizaciones por peso:

| Estado de carga | Efecto core |
|---|---|
| Carga ligera | Sin penalización. |
| Carga media | Sin penalización normal; puede afectar sigilo si la escena lo justifica. |

- Compañeros, caravanas, asentamientos y contenedores seguros permiten mover carga sin llevarla encima.

CAPITULO 15 — GLOSARIO Y TABLAS MAESTRAS
======================================================================

Glosario mínimo:

| Término | Definición |
|---|---|
| AP | Puntos de Acción. Recurso para actuar en turno. |
| Nodo | Punto táctico del mapa ocupable o recorrible. |
| Submapa | Red de nodos dentro de una ubicación. |
| Ubicación | Punto grande del mundo: ciudad, ruina, vault, asentamiento, ruta o edificio. |
| Mapamundi | Red de ubicaciones conectadas por rutas. |
| Cobertura | Objeto o posición que dificulta, reduce o bloquea ataques. |
| Elevación | Condición de altura; da ángulo, visión o alcance, pero no cobertura automática. |
| Línea de visión | Condición que permite ver y atacar desde un nodo a otro. |
| Parte del cuerpo | Zona impactada por un ataque: cabeza, torso, brazo izquierdo, brazo derecho, pierna izquierda o pierna derecha. |
| Perk | Ventaja o habilidad especial obtenida por progresión. |
| Momentum | Éxitos sobrantes después de superar la dificultad de una tirada 2d20. |
| Daño bruto | Daño base del arma antes de restar cobertura, armadura o resistencias. |
| Daño final | Daño que queda después de aplicar cobertura, defensa de pieza, resistencias y bonos/reducciones. |
