TOMO 07 — ASENTAMIENTOS: PUENTE CON SUPERVIVENCIA Y MODO LIBRE

1. PUENTE ASENTAMIENTO ↔ SUPERVIVENCIA
Un asentamiento seguro puede reducir tiradas de descanso, enfermedad, guardia y clima. Para contar como descanso completo debe tener:
- cama o refugio,
- agua,
- comida o racion,
- defensa minima,
- ausencia de ataque activo.

2. CAMPAMENTO VS ASENTAMIENTO
| Elemento | Campamento | Asentamiento |
|---|---|---|
| Duracion | temporal | permanente |
| Guarda objetos | limitado | si |
| Colonos | no | si |
| Defensa | guardia/trampa simple | sistema completo |
| Produccion | no | comida, agua, recursos |
| Riesgo | ataque/saqueo | ataque de asentamiento |

3. EVENTOS DE ASENTAMIENTO PARA MODO LIBRE
1) ataque raider,
2) falta de agua,
3) enfermedad,
4) caravana perdida,
5) sabotaje de faccion,
6) solicitud de rescate,
7) terminal bloqueado,
8) disputa interna,
9) criatura cercana,
10) oportunidad comercial.

## REGLAS PRÁCTICAS DE ASENTAMIENTOS


- Agua: producir más de la necesaria genera agua purificada en el taller.
- Comida: cada colono consume 1 unidad aproximada; asignar colonos a cultivos.
- Defensa: intentar que defensa >= comida + agua para reducir ataques.
- Camas: una cama por colono, bajo techo si se busca felicidad alta.
- Felicidad: sube con comida, agua, defensa, camas, tiendas, clínicas, decoración y ausencia de problemas.
- Líneas de suministro: usar Local Leader para compartir chatarra entre talleres.
- Artillería Minutemen: útil en asentamientos exteriores, requiere colono asignado.
- En Survival, distribuir bases con cama, agua, médico y almacenamiento reduce riesgo de viaje.

Orden mínimo de construcción:
1. Taller asegurado.
2. Camas bajo techo.
3. Agua.
4. Comida.
5. Defensa igual o superior a comida + agua.
6. Energía solo donde desbloquee agua, luces, radio, tiendas o defensas.
7. Ruta de suministro cuando el asentamiento deje de ser aislado.
8. Comercio, clínica, artillería o decoración cuando la supervivencia esté cubierta.


## MÓDULO DE ASENTAMIENTOS PARA 2D20


Este módulo convierte cada asentamiento en un nodo de campaña. No reemplaza las reglas base: sirve para resolver en mesa construcción, crisis, rutas y economía sin depender del videojuego.

Escala de estado:
| Estado | Condición | Efecto en mesa |
| --- | --- | --- |
| Precario | Falta comida, agua, camas o defensa | +1 complicación en escenas sociales y de defensa del asentamiento. |
| Estable | Necesidades básicas cubiertas | Funciona como refugio seguro, taller y punto de descanso. |
| Próspero | Tiene comercio, clínica, excedente o ruta activa | Genera botín menor, rumores, contactos o ingresos entre sesiones. |
| Fortificado | Defensa alta, muros, torretas o artillería | Reduce la gravedad de ataques y permite resistir asedios narrativos. |
| Especializado | Cumple una función clara: agua, comida, robots, comercio, vault, puerto | Otorga ventaja narrativa cuando esa función importa en una misión. |

Chequeos de mejora:
| Mejora | Prueba sugerida | Materiales clave | Fallo o complicación |
| --- | --- | --- | --- |
| Reparar infraestructura | INT + Reparación | Acero, madera, tornillos | Algo queda inestable; el DJ introduce una avería futura. |
| Instalar energía | INT + Ciencia o Reparación | Cobre, circuitería, aceite | Cortocircuito, ruido, consumo extra o piezas quemadas. |
| Levantar defensas | STR + Supervivencia o INT + Reparación | Acero, hormigón, engranajes | Punto ciego, munición escasa o torreta mal ubicada. |
| Organizar población | CHA + Discurso | Comida, camas, seguridad | Rivalidad interna, queja pública o colono que se marcha. |
| Abrir comercio | CHA + Trueque | Chapas, madera, acero | Deuda, proveedor dudoso o impuesto de facción. |
| Crear clínica | INT + Medicina | Antiséptico, vidrio, electricidad | Falta de medicamentos o paciente con secreto peligroso. |

Reloj de crisis:
Usa un reloj de 4 segmentos para problemas locales y uno de 6 segmentos para asedios o tramas de facción. Cada fallo, viaje largo ignorado o ataque no resuelto marca 1 segmento. Al llenarse, el asentamiento pierde un estado, un recurso, un NPC o una ruta.

Producción entre sesiones:
| Tipo de asentamiento | Produce si está estable | Si está próspero |
| --- | --- | --- |
| Granja / comida | Comida, rumores rurales | Adhesivo, semillas, apoyo de colonos |
| Agua / costa | Agua purificada, pesca, rutas | Ingresos, medicinas, contacto mercante |
| Comercio | Chapas, información, encargos | Objeto raro, descuento o caravana aliada |
| Fortaleza / facción | Seguridad, munición, patrulla | Artillería, escolta o favor militar |
| Técnica / robots | Piezas, reparaciones | Robot auxiliar, sensor o mod tecnológico |
| Vault / interior | Refugio, energía, ciencia | Experimento, clínica avanzada o vivienda masiva |
| Casa urbana / refugio | Descanso, almacenamiento, contactos | Rumor de ciudad, comprador, abogado o acceso social |

## CRUCES CON OTROS TOMOS


- Ubicaciones: usar el tomo de ubicaciones para rutas, mapas, biomas y peligros cercanos.
- Bestiario: seleccionar enemigos por bioma y facción; no usar la misma amenaza para todos los asentamientos.
- Armas: las defensas y guardias deben reflejar economía local; granjas usan armas comunes, fortalezas usan rifles/pesadas.
- Armaduras/equipo: ropa social para comercio, armadura de facción para control político, servoarmadura para defensa militar.
- Settlers Supplement: usar sus reglas y opciones como cantera compatible, pero adaptar todo lo que aumente complejidad administrativa o desplace el foco de la campaña.

  - Cama, contenedores separados, banco de trabajo, iluminación, botiquín y zona de reunión.

Línea de suministro:
  - No admite ruta clásica de suministro; tratar como nodo personal conectado por comercio, mensajeros o favores.

Escena inicial:
  - Un vecino, guardia o comerciante nota que los PJ ocupan Home Plate y trae una oferta, queja o advertencia sobre Diamond City.

Problema local:
  - La propiedad es segura, pero todo lo que entra o sale deja rastro: compras raras, armas pesadas, synths refugiados o cargamentos de chatarra pueden llamar la atención.

Producción y uso:
  - Estable: descanso, almacenamiento seguro, acceso a tiendas y rumores urbanos.
  - Próspero: comprador fiable, contacto médico, informante, abogado, periodista o intermediario de mercado negro.

Eventos sugeridos:
  - Chantaje por un objeto guardado.
  - Guardia pide revisar el lugar.
  - Comerciante ofrece pieza rara.
  - Refugiado synth o informante pide esconderse una noche.
  - Una facción intenta usar Home Plate como buzón muerto.

Cruces de campaña:
  - Misiones asociadas: Diamond City Blues, In Sheep's Clothing, historias de Piper/Nick, tramas de synths e investigaciones urbanas.
  - Bestiario sugerido: Seguridad humana, mafiosos, infiltrados synth, cazadores, asesinos contratados.
  - Equipo recomendado: cerraduras, luces, cama, botiquín, cajas fuertes, armas cortas, ropa social y documentos falsos.
  - Uso en mesa: Refugio urbano, base de investigación, sala de consejo, archivo de pistas y punto de compraventa.


## APÉNDICE - VAULTS COMO ASENTAMIENTOS O PUESTOS


- Otros Vaults pueden funcionar como puestos narrativos, bases temporales o enclaves de facción si la mesa lo permite.
- Reglas recomendadas: limpiar amenaza, asegurar energía, agua, camas, ruta de acceso y facción administradora.
